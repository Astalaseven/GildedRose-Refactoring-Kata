# gilded-rose-kata

gilded-rose-kata is a kata exercise about refactoring.
Source: https://github.com/emilybache/GildedRose-Refactoring-Kata

## Test

Use gradle to run the tests:

```bash
gradlew clean test
```

## Build

```bash
gradlew clean build
```
